## Design Guidelines

- **Color palette**: Dark navy (#0A0E17), Gold (#D4AF37), Purple (#8B5CF6), Blue (#3B82F6)
- **Style**: Premium casino, dark glass with gold accents
- **Icon style**: Flat/outlined with subtle gradients, consistent 2px stroke weight
- **Format**: PNG with transparency, @3x for Retina
- **Naming**: snake_case, prefix by location (e.g., `tab_`, `icon_`, `quick_`, `game_`)

## How to Add Assets

1. Place files in `mobile/assets/icons/` or `mobile/assets/game/` as appropriate
2. Update the `A = { ... }` asset object in the screen file that uses it
3. Replace the placeholder `<Text style={styles.iconEmoji}>🎰</Text>` with `<Image source={A.newAsset} />`
4. Remove the `🔲` status and mark as `✅` in this file
